# DeusFin — Макеты (design previews)

Витрина дизайн-макетов продукта DeusFin. Каждый макет — самодостаточный HTML-прототип, открывается в браузере без сборки и зависимостей. Данные демонстрационные.

**Витрина (GitHub Pages):** ссылка появится после включения Pages — `https://<account>.github.io/deusfin-mockups/`

## Структура
- `index.html` — витрина со списком всех макетов (живые превью).
- `mockups/` — сами макеты, по одному HTML на экран.

## Как добавить новый макет
1. Положи `*.html` в `mockups/` (например `02-opiu.html`).
2. В `index.html` скопируй один блок `<a class="card">…</a>` и поправь ссылку, заголовок, описание.
3. `git add . && git commit -m "add: <экран>" && git push` — витрина обновится автоматически.

## Доступ
Репозиторий публичный — витрина открывается по ссылке кем угодно. Участники проекта, добавленные как collaborators, могут заливать новые макеты.
